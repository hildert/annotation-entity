# annotation-entity
A PHP Library help phper convert array to dto entity.

### Setup

```txt

$ git clone git@github.com:hildert/annotation-entity.git
$ cd annotation-entity
$ composer install
$ cp phpunit.xml.dist phpunit.xml

# Run Tests
$ vendor/bin/phpunit
```
